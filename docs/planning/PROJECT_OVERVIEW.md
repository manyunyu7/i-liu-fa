# DuoManifest - Project Overview

## Vision
A Duolingo-styled web application focused on personal growth through:
- **Law of Affirmation** - Daily affirmations with streak tracking
- **Bucket List** - Track life goals and achievements
- **Dreams** - Visualize and manifest dreams
- **Planner** - Daily/weekly planning with gamification

## Core Philosophy
Gamify personal development using proven engagement mechanics from Duolingo:
- Daily streaks and XP points
- Achievement badges and levels
- Progress visualization
- Friendly reminders and notifications
- Social features (optional sharing)

## Target Users
- Self-improvement enthusiasts
- Goal-oriented individuals
- People practicing manifestation/LOA
- Anyone wanting to track life goals

## Tech Stack
- **Backend**: Laravel 12 (PHP 8.3+)
- **Frontend**: Blade + Tailwind CSS + Alpine.js
- **Database**: MySQL/SQLite
- **Authentication**: Laravel Breeze/Sanctum
- **Styling**: Duolingo-inspired design system

## Key Features

### 1. Affirmations Module
- Daily affirmation practice
- Custom affirmation library
- Category-based affirmations (wealth, health, love, success)
- Voice recording for affirmations
- Streak tracking

### 2. Bucket List Module
- Create life goals
- Categorize by life area
- Progress tracking (0-100%)
- Photo/memory attachments
- Completion celebrations

### 3. Dreams Module
- Dream journal
- Vision board digital
- Manifestation tracking
- Dream categories

### 4. Planner Module
- Daily intentions
- Weekly goals
- Monthly reviews
- Habit tracking integration

### 5. Gamification System
- XP points for activities
- Daily/weekly streaks
- Achievement badges
- Levels and progression
- Leaderboards (optional)

## Design Principles
- Clean, friendly UI (Duolingo aesthetic)
- Green as primary color
- Rounded corners, soft shadows
- Mascot character (owl-like for manifestation)
- Celebratory animations
- Mobile-first responsive design
